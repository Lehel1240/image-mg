Első lépések

Alvid használata rendkívül egyszerű. Kövesd az alábbi lépéseket a beszélgetés megkezdéséhez:

  1. Navigálj a Csevegés oldalra a menüből.
  2. Írd be az üzenetedet az oldal alján található szövegmezőbe.
  3. Nyomd meg az Entert vagy kattints a küldés gombra.
  4. Várd meg Alvid válaszát, ami néhány másodpercen belül megjelenik.

Példa parancsok és kérdések

Nem tudod, hogyan kezdd? Íme néhány ötlet, amivel próbára teheted Alvid képességeit:

Egyszerű kérdések:

"Mi Magyarország fővárosa?"

Kreatív írás:

"Írj egy rövid verset az őszről."

Programozási segítség:

"Adj egy Python kódrészletet, ami beolvas egy JSON fájlt."

Összefoglalás:

"Foglald össze a relativitáselmélet lényegét három mondatban."

API Hozzáférés

A chatbot funkcionalitása egy egyszerűsített API végponton keresztül is elérhető, ami ideális lehet fejlesztői integrációkhoz. A kommunikáció JSON formátumban történik.

  • Végpont: /api/chat
  • Metódus: POST
  • Request Body (JSON):
// A kérés törzsének egy JSON objektumnak kell lennie
{
  "message": "Szia, hogy vagy?"
}
  • Response Body (JSON):
// Sikeres válasz esetén
{
  "reply": "Szia! Köszönöm, jól vagyok. Miben segíthetek?"
}
